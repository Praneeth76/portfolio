# Portfolio Website

This is a personal portfolio website built using the MERN stack (MongoDB, Express.js, React, Node.js). The website showcases my projects, skills, and experience in a clean, dynamic, and interactive interface. It is deployed on Vercel for easy access.

## Features

- **Project Showcase**: Display of all major projects and their descriptions.
- **Skills & Experience**: Interactive sections highlighting key technical skills and professional experience.
- **Dynamic Content**: Content is fetched from a backend server using the MERN stack, making it easy to update and manage.
- **Responsive Design**: The portfolio is fully responsive, ensuring a great user experience across all devices.
- **Contact Form**: A simple contact form to get in touch with me.

## Tech Stack

- **MongoDB**: NoSQL database for storing dynamic content like projects and skills.
- **Express.js**: Backend web framework to handle API requests and server-side logic.
- **React.js**: Frontend library for building the user interface and managing dynamic content.
- **Node.js**: JavaScript runtime to run the server-side code.
- **Vercel**: Hosting platform for easy deployment of the website.
